## A. Filing route and sequence
1. **Provisional** (Form 1 + Form 2-provisional + Form 3 + Form 5 optional at provisional + Form 28 if small entity/startup). Secures priority date cheaply.
2. Within **12 months**: **Complete specification** (Form 2-complete with claims from the spec + drawings from `figures.md`).
3. **Form 18A** expedited examination (eligible: natural person / startup / small entity).
4. Optional **Form 9** early publication (shortens the 18-month wait to ~1 month).
5. Respond to the First Examination Report (FER) within the statutory period using the §7 technical-effect arguments and §7A claim chart.
